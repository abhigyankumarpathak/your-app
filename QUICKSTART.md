# Quick Start Guide - Focus App

## 🚀 First Time Setup

### 1. Install Dependencies
```bash
cd /Users/amitpathak/Documents/python/your-app
npm install
```

### 2. Run on Your Device

**For iPhone:**
```bash
npm run ios
```

**For Android:**
```bash
npm run android
```

### 3. Grant Permissions
- When prompted, tap **"Allow"** for calendar/health permissions
- This enables all tracking features

### 4. Customize Your App
1. Open **Settings** (swipe drawer → Settings)
2. Choose your **Theme** (Light/Dark/Warm)
3. Pick your **Accent Color** (Blue, Green, Purple, etc.)
4. Adjust **Text Size** if needed
5. Toggle **Animations** for your preference

## 🔧 Troubleshooting

### Data not saving?
```bash
# Clear app data
1. Settings → Apps → Focus → Clear Data
2. Restart app
```

### App not responding?
```bash
# Restart the app
1. Close the app completely
2. Wait 5 seconds
3. Reopen it
```

### Need to clear all data?
```bash
# This will reset everything
1. Go to Settings
2. Scroll to app version
3. Clear all data from within the app
```

### Want to reinstall?
```bash
# Clean reinstall
npm run android --clean  # or npm run ios --clean
```

## 📊 Understanding Your Stats

### Study Statistics
```
Sessions Today: 3
   = 3 separate study blocks you've started

Total Time: 1h 45m
   = Sum of all your study session durations

Avg Session: 35m
   = Total time ÷ number of sessions
```

### Wellness Statistics
```
Avg Sleep: 7.2h
   = Average of all logged sleep durations

Avg Screen: 4.1h
   = Average of all logged screen times

Total Logs: 15
   = How many days you've logged data
```
